# cloudnative 云原生框架
这里参考了 google-cloud-go

对 micro v2 / v3 等的依赖，都隐藏在 Framework 中

## Overview
microbase 提供了云服务所需的关键功能. 
microbase 负责处理分布式系统的复杂性, 提供一个简单的编程模型给应用
